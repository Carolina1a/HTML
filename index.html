PROTECCIÓN DE DATOS PERSONALES

A continuación, se informa el tratamiento que YANBAL le dará a la información que usted envíe a través de formularios para completar datos en el Sitio Web y/o en la Tienda Virtual de YANBAL, con el fin de que pueda decidir si desea ingresar sus datos o no. Los datos personales que usted registre voluntariamente en este Sitio Web y/o en la Tienda Virtual de YANBAL serán incorporados en las bases de datos que administra YANBAL y serán tratados para el adecuado desarrollo de su relación comercial con YANBAL, para fines estadísticos, comerciales y o de marketing, para conocer sus preferencias y adaptar el Sitio Web y la Tienda Virtual a sus gustos y necesidades, así como para el desarrollo de estrategias comerciales, publicitarias y actividades promocionales y para el envío de información comercial relacionada con los productos Yanbal y con las actividades de mercadeo en red o multinivel.
